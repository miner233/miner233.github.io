# Python if 语句
if 语句是对条件判断的语句，根据判断结果运行相应代码。

## 基本语法
```Python
if ......:   # 如果......运行......
    ......
elif ......:   # 否则如果......运行......
    ......
else:   # 否则运行......
    ......
```

## 温馨提示
不要忘记":"！

## 代码示例
代码时间到！！！
```Python
name="卞晓涵"
if name=="卞晓涵":
    print("姓名：卞晓涵")
    print("年龄：114514")
    print("属性：一个大傻逼")
elif name=="季不惠":
    print("姓名：季不惠")
    print("年龄：114514")
    print("属性：丧心病狂")
else:
    print("姓名：",name)
    print("年龄：无数据")
    print("属性：无数据")
```
